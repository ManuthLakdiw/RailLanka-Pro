System User Manual

RailLanka Pro System Overview

Comprehensive railway management solution for operations across stations, trains, schedules, and personnel.

Common Questions:

  • How to add a new train: Navigate to "Manage Trains" → "Add New Train" → Fill in details
  • Modify schedules: Go to "Manage Schedule" → Edit icon → Make changes → Save
  • Access issues: Check permissions with admin or contact IT support with error message
  • Generate reports: Access "Reports" → Select type → Set parameters → Generate

FAQ & Troubleshooting

Common Issues & Solutions:

Login Problems:

"Invalid credentials" error

→ Ensure Caps Lock is off, use password reset, contact admin if locked

System Performance:

Slow response times

→ Clear browser cache, check internet, try off-peak hours

Data Entry Errors:

Cannot save form data

→ Check required fields, validate formats, avoid duplicates

Quick Troubleshooting Steps:

  1. Refresh the page (F5)
  2. Clear browser cache and cookies
  3. Try a different browser (Chrome/Firefox)
  4. Check internet connection
  5. Verify JavaScript is enabled
